sdcflows.workflows package¶

Fieldmap estimation and unwarping workflows.

Automatic selection of the appropriate susceptibility-distortion correction (SDC) method¶

If the dataset metadata indicate that more than one field map acquisition is IntendedFor (see the BIDS Specification), the following priority will be used:

  1. sdc_pepolar (or blip-up/blip-down)

  2. sdc_direct_b0

  3. sdc_phasediff

  4. sdc_fieldmapless
